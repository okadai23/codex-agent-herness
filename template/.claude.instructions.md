# Claude Code project instructions

このファイルは Claude Code 利用時の最小運用ガイドです。

## 基本方針
- 既存のテスト・lint・format コマンドを優先して実行する。
- generated file は直接編集せず、生成元（OpenAPI / template / script）を更新する。
- 秘密情報は環境変数で扱い、リポジトリへ保存しない。

## 推奨検証コマンド
- TypeScript系: `pnpm verify:fast`
- API契約: `pnpm api:verify`
- Docs: `pnpm docs:verify`
- Python系: `uv run pytest`

## 禁止/注意
- 破壊的コマンド（`rm -rf`, `git push --force` など）は実行しない。
- CIで再現できないローカル専用手順を最終成果物にしない。
