# $notion-mcp-read

## 目的
Notion MCP を使って Notion ワークスペースから安全に情報を取得する。

## 前提
- Notion Integration が作成済みで、対象ページ/データベースへアクセス権が付与されている。
- MCP サーバー設定が完了し、認証トークンが環境変数で読み込まれている。

## 手順
1. 取得対象（ページ / データベース / プロパティ）を明確化する。
2. 必要最小限のクエリで情報を取得する。
3. 出力には機密情報が含まれていないか確認する。
4. 取得結果の出典（Notion URL/ページ名）を記録する。

## 追加ルール
- トークンや機密値をログ・コミットへ含めない。
- 大量取得が必要な場合はフィルター条件を先に適用する。
