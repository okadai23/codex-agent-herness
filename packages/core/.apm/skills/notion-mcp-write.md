# $notion-mcp-write

## 目的
Notion MCP を使って Notion に追記・更新を行い、作業ログや仕様を同期する。

## 前提
- Notion Integration に書き込み権限が付与されている。
- 書き込み先ページ/データベースのスキーマを事前に確認している。

## 手順
1. 更新対象（ページID / DB行）と変更内容を明確化する。
2. 変更前の重要フィールドを読み取り、上書き事故を防ぐ。
3. 最小単位で書き込みを実行する。
4. 書き込み結果を再読込して反映を検証する。

## 追加ルール
- 破壊的変更（大量削除・全件更新）は避け、必要時は段階的に実施する。
- 監査可能性のため、更新理由を Notion 側へ短く残す。
